# flask-angular

Run commands 
```
python -m venv venv
```
```
cd venv/Scripts
```
```
activate.bat
```
```
cd ../../scripts
```
```
init.bat
```
to install requirements, go to movies folder and run:
```
pip install -r requirements.txt
```
### To start application
Go to back-end and run
```
flask run
```
